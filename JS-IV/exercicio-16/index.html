Aprendendo a utilizar bibliotecas de terceiros com o módulo dayjs

Introdução

Neste vídeo, vamos aprender uma habilidade muito importante para qualquer programador: a capacidade de utilizar bibliotecas de terceiros.

Bibliotecas de terceiros são bibliotecas desenvolvidas por terceiros, ou seja, não foram desenvolvidas por nós. Elas contêm funcionalidades prontas que podemos utilizar em nossos projetos para aumentar nossa produtividade e evitar reinventar a roda.

Vamos praticar o uso de bibliotecas de terceiros através do módulo dayjs, que é uma biblioteca JavaScript moderna para parse, validação, manipulação e formatação de datas.

O desafio

O desafio proposto é:

Criar uma função JavaScript que recebe como parâmetro a data de nascimento de uma pessoa no formato string e mostre no terminal:

  • A idade dessa pessoa no dia atual (hoje)
  • Sua próxima data de aniversário no formato DD/MM/YYYY
  • Quantos dias faltam para o seu próximo aniversário

Para isso, vamos precisar:

  • Acessar o site do NPM e pesquisar sobre o módulo dayjs
  • Consultar a documentação do dayjs
  • Aprender a utilizá-lo para resolver o desafio proposto

Conhecendo o NPM e o dayjs

O NPM (Node Package Manager) é o gerenciador de pacotes do Node.js. Ele permite instalar e gerenciar bibliotecas e módulos JavaScript de maneira fácil.

Basta acessar o site npmjs.com, pesquisar pelo módulo "dayjs" e veremos informações sobre ele, como quantidade de downloads, versão atual, links para documentação e homepage, além de um guia de como utilizá-lo.

Podemos ver que o dayjs é um módulo popular, com mais de 19 milhões de downloads por semana! Isso significa que muitos desenvolvedores estão utilizando essa biblioteca em seus projetos.

Além disso, na seção "dayjs docs" temos um link direto para a documentação oficial do módulo, onde encontraremos guias e exemplos detalhados de uso.

Lendo a documentação

A documentação é essencial para aprender a usar qualquer biblioteca de terceiros. Ela explica como instalar, configurar e utilizar os recursos disponíveis no módulo.

Ao acessar a documentação do dayjs, vemos logo de início um guia de instalação. Para este projeto, como estamos utilizando Node.js, basta instalar via NPM:

npm install dayjs

Em seguida, para utilizá-lo no código:

var dayjs = require('dayjs') dayjs().format() // 2023-02-08T15:03:33+08:00

Perceba que usamos require() porque estamos no Node.js. Se estivéssemos no browser, usaríamos import dayjs from 'dayjs'.

A documentação explica os principais métodos da API do dayjs, com vários exemplos de uso. Por isso, é muito importante ler e entender esses conceitos antes de sair codificando.

Resolvendo o desafio

Agora que já conhecemos o dayjs e sabemos como utilizá-lo, podemos começar a implementar a função solicitada no desafio.

A dica deixada é procurar na documentação métodos para:

  • Adicionar intervalos de tempo a uma data
  • Calcular a diferença entre duas datas
  • Exibir uma data em um formato específico

Então, depois de ler a documentação, identificamos os seguintes métodos úteis:

  • dayjs(): cria um objeto Dayjs com a data e hora atuais
  • dayjs(dataString): cria um objeto Dayjs a partir de uma string de data
  • dayjs().add(numero, 'unidadeTempo'): adiciona uma quantidade de tempo
  • dayjs().diff(outraData): retorna a diferença em milissegundos entre duas datas
  • dayjs().format('formato'): formata a data conforme especificado

Portanto, nossa função poderia ser implementada assim:

const dayjs = require('dayjs')function calculaIdade(dataNascimento) {  const hoje = dayjs()    const nascimento = dayjs(dataNascimento)    const idade = hoje.diff(nascimento, 'years')  const proximoAniversario = nascimento.add(idade + 1, 'years')  const diasParaAniversario = hoje.diff(proximoAniversario, 'days')  return {    idade,    proximoAniversario: proximoAniversario.format('DD/MM/YYYY'),    diasParaAniversario      }}console.log(calculaIdade('1990-02-03'))

Essa função recebe a data de nascimento como string, calcula a idade comparando com a data atual, encontra a próxima data de aniversário adicionando a idade + 1 ano no objeto data de nascimento, calcula quantos dias faltam para essa data e retorna os dados formatados conforme solicitado!

Dessa forma, conseguimos completar o desafio de maneira simples e rápida, aproveitando uma excelente biblioteca open source.

Considerações finais

Aprender a usar bibliotecas e módulos de terceiros é uma habilidade essencial para qualquer desenvolvedor moderno. Isso porque evita termos que desenvolver soluções do zero para problemas comuns, como manipulação de datas.

O NPM possui milhares de bibliotecas úteis que podemos aproveitar em nossos projetos. Basta procurar por aquela que resolve nosso problema específico.

Porém, para utilizar uma biblioteca de forma efetiva, é absolutamente necessário ler sua documentação. Nela encontraremos guias de instalação e os detalhes de cada recurso disponível na API.

No caso do dayjs, vimos que se trata de uma excelente opção para lidar com datas no JavaScript moderno de forma simples e intuitiva. Seu uso é bem parecido com a nativa Date API, com a vantagem de ter uma interface mais amigável e imutável.

Portanto, não hesite em aproveitar bibliotecas de terceiros em seus projetos! Elas ajudarão você a resolver problemas complexos de maneira rápida e mantendo seu código limpo e descomplicado.
