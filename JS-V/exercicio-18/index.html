Orientação a Objetos e Classes em JavaScript

Antes de criarmos nossas classes, precisamos entender alguns conceitos básicos:

Objetos: são entidades independentes com propriedades e tipos. Em JavaScript, objetos podem ser criados utilizando a notação literal de objeto {} ou com o construtor Object().

Propriedades: são as características que definem um objeto, como por exemplo, um nome, uma cor, um tamanho, etc. As propriedades são definidas na forma de pares chave-valor, como nome: "João" por exemplo.

Métodos: são ações que podem ser executadas em objetos. Os métodos são definidos na forma de funções dentro do objeto.

Classes: são modelos que servem como blueprint para a criação de objetos. As classes definem as propriedades e métodos compartilhados por todos os objetos criados a partir dela.

Construtores: são métodos especiais utilizados para inicializar objetos criados a partir de uma classe. O construtor é chamado no momento da criação do objeto utilizando a palavra chave new.

Instanciação: é o ato de criar um objeto novo baseado em uma classe. Ao instanciar uma classe, o construtor é chamado para inicializar as propriedades do novo objeto.

Agora que revisamos os conceitos principais, podemos começar a implementar nossas duas classes em JavaScript!

Implementando a Classe User

A primeira classe que precisamos criar é a User, que representa um usuário com as seguintes características:

  • fullName - O nome completo do usuário
  • email - O email do usuário
  • password - A senha do usuário
  • login() - Um método para realizar o login

Começamos criando um arquivo separado chamado User.js e dentro dele definimos a classe:

class User {  constructor(fullName, email, password) {    this.fullName = fullName;    this.email = email;     this.password = password;  }    login(email, password) {    if (email === this.email && password === this.password) {      console.log('Login realizado com sucesso!');    } else {      console.log('Email ou senha incorretos.');    }  }}

Vamos explicar parte por parte:

  • Utilizamos a palavra chave class para definir a classe User.

  • Dentro dela, definimos o método especial constructor() que será chamado na hora da instanciação.

  • No construtor, recebemos os parâmetros fullName, email e password que serão as propriedades do objeto.

  • Utilizamos o this para fazer referência ao objeto que está sendo construído e armazenar essas propriedades nele.

  • Depois, definimos o método login(), que recebe um email e uma senha para verificar se batem com as propriedades do objeto.

  • Dentro do login, comparamos os parâmetros com as propriedades do objeto utilizando this e retornamos uma mensagem de sucesso ou falha no login.

Agora, podemos testar nossa classe User instanciando alguns objetos:

const joao = new User('João Silva', 'joao@email.com', '1234');joao.login('joao@email.com', '1234'); // Login realizado com sucesso! const maria = new User('Maria Oliveira', 'maria@email.com', '5678');maria.login('maria@email.com', '1234'); // Email ou senha incorretos

Perfeito! Nossa classe User está funcionando corretamente.

Implementando a Classe Product

A segunda classe que precisamos implementar é a Product, que representa um produto com as seguintes características:

  • name - O nome do produto
  • description - A descrição do produto
  • price - O preço do produto
  • inStock - A quantidade em estoque, que começa em 0
  • addStock() - Um método para adicionar unidades ao estoque
  • calculateDiscount() - Um método que aplica um desconto ao preço

Vamos criar um arquivo Product.js e implementar:

class Product {  constructor(name, description, price) {    this.name = name;    this.description = description;    this.price = price;     this.inStock = 0;   }    addStock(quantity) {   this.inStock += quantity;   }    calculateDiscount(percentage) {    let discount = this.price * (percentage / 100);    return this.price - discount;  }}

Novamente, vamos explicar parte por parte:

  • Definimos a classe Product com o construtor recebendo name, description e price.

  • Também inicializamos a propriedade inStock com o valor padrão 0.

  • O método addStock() recebe uma quantidade e soma ao estoque atual utilizando this.inStock.

  • Já o método calculateDiscount() recebe uma porcentagem de desconto, calcula sobre o preço e retorna o preço com o desconto aplicado.

Para testar:

const produto = new Product('Camisa', 'Camisa branca', 50);produto.addStock(10); console.log(produto.inStock); // 10console.log(produto.calculateDiscount(25)); // 37.5 (25% de desconto)

Pronto! Nossas duas classes estão implementadas com sucesso seguindo os requisitos passados.
